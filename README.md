## Building and Running
To build both Client and Server:
```shell
git clone https://github.com/Tom-stack3/AP1-ex3.git
cd AP1-ex3
mkdir build
cd build
cmake ..
make
```

### Running the Server
Go to `./build/server` folder.
Inside run the following:
```shell
// Running the server
./Server
```
### Running the Client
Go to `./build/client` folder.
Inside run the following:
```shell
// Running a client
./Client
```
